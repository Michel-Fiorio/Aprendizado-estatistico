Realizaremos anlise do banco de dados Breast Cancer utilizando as tcnicas de Anlise Discriminante Linear e Anlise Discriminante Quadrtica. Poderemos verificar qual tcnica apresentar modelos com melhores resultados de classificao.

Anlise Discriminante Linear:

library(MASS)

discrilinear.bc = lda(Class ~ ., data = bc[,2:11], prior = c(1,1)/2 , CV = TRUE)

# Obter a classificao resultante do modelo para o conjunto de dados

Classif.linear = discrilinear.bc$class
# Gerar a matriz de confuso do modelo
confusao.linear = table(bc$Class, Classif.linear)
confusao.linear
##            Classif.linear
##             benign malignant
##   benign       436         8
##   malignant     18       221
# Calculando taxa de erro aparente do modelo

tea.linear = 1 - (sum(diag(confusao.linear))/sum(confusao.linear))
tea.linear
## [1] 0.03806735

Anlise discriminante Quadrtica:

discriquad.bc = qda(Class ~ ., data = bc[,2:11], CV = TRUE)

# Obter a classificao resultante do modelo para o conjunto de dados

Classif.quad = discriquad.bc$class
# Gerar a matriz de confuso do modelo

confusao.quad = table(bc$Class, Classif.quad)
confusao.quad
##            Classif.quad
##             benign malignant
##   benign       416        28
##   malignant      6       233

## [1] 0.04978038

Anlise dos resultados: realizamos a anlise discriminante dos dados no formato linear e quadrtico. Utilizamos na gerao dos modelos a validao cruzada do tipo leave-one-out. Foi gerada a matriz de confuso e calculado a taxa de erro aparente dos modelos. Pudemos observar que a taxa de erro foi muito pequena, de apenas 3,8% no modelo da tcnica linear e de 4,97% no modelo da tcnica quadrtica. Portanto o modelo discriminante linear foi mais eficiente na classificao dos dados.
